Lesson 2: Sets

1. Introduction to Sets

A set is a collection that is unordered, unindexed, and does not allow duplicates.

Feature Set List
Syntax {} []
Ordered No Yes
Duplicates Not allowed Allowed 
# Set creation
daily_tasks = {"wake up", "brush teeth", "go to school"}
print(daily_tasks)

Output:

{'wake up', 'brush teeth', 'go to school'}

2. Accessing Set Items

Since sets are unordered, you can't access items by index. Use these methods:

Method Description Example
for loop Iterate through all items for x in my_set:
in operator Check if item exists if "item" in my_set: 
fruits = {"apple", "banana", "cherry"}
for fruit in fruits:
    print(fruit)
fruits = {"apple", "banana", "cherry"}
print("banana" in fruits)

Output: True

3. Set Methods

Method Description Example
add() Adds an element my_set.add("item")
remove() Removes an element my_set.remove("item")
clear() Removes all elements my_set.clear()
del Deletes the set completely del my_set 
colors = {"red", "blue"}
colors.add("green")
colors.remove("red")
print(colors)

Output: {'blue', 'green'}

numbers = {1, 2, 3}
numbers.clear()  # Empty set
del numbers      # Deletes the set
